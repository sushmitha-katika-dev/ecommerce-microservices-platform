# Database Schemas

Schemas are automatically generated by Hibernate's `ddl-auto` or managed via Flyway/Liquibase scripts located in the `init-sql` directory of the repository.

- `users`: id, email, password_hash, first_name, last_name, status, created_at, updated_at
- `products`: id, category_id, name, sku, description, price, active, created_at, updated_at
- `carts`: id, user_id, session_id, status, created_at, updated_at
- `orders`: id, user_id, status, total_amount, shipping_address, created_at, updated_at

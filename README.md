# E-Commerce Microservices Platform

Welcome to the official documentation for the E-Commerce Microservices Platform. This repository is built using Java Spring Boot and follows a strict microservices architecture, emphasizing decoupling, event-driven communication via Kafka, and security using JWT.

## Tech Stack
- Java 17
- Spring Boot 3.x
- Spring Cloud Gateway
- Spring Security (JWT)
- Apache Kafka
- MySQL / PostgreSQL (Database per service)
